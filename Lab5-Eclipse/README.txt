To compile and run this program on state capital machines

$ cd src
$ javac -cp "./:/usr/share/java/mysql-connector-java.sql.jar" GUI.java Library.java Lab5.java

$ java -cp "./:/usr/share/java/mysql-connector-java.sql.jar" Lab5

If you wish to not compile, you can run the jar located in the build folder.

$ cd build
$ java -jar Lab5-StateCapitals.jar
